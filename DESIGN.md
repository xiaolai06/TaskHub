# TaskFlow+ 设计文档

## 色彩系统
- 主色：紫色系（#5B5FC7 / OKLCH 适配）
- 成功：绿色系
- 警告：橙色系
- 错误：红色系
- 中性色：偏紫的灰色（chroma 0.005-0.01）

## 主题
浅色模式为主，深色模式可选。

场景：一人公司老板在办公室电脑上使用，白天自然光环境。

## 字体
- 标题：系统字体栈（Segoe UI / PingFang SC / Microsoft YaHei）
- 正文：14-15px，行高 1.8
- 代码：等宽字体

## 间距节奏
- 大区块间距：32px
- 组件间距：16-20px
- 内部间距：12-16px
- 紧凑间距：8px

## 组件风格
- 卡片：圆角 10px，轻微阴影
- 按钮：圆角 8px，hover 状态
- 输入框：圆角 8px，focus 紫色边框
- 表格：斑马纹，hover 高亮

## 布局
- 侧边栏固定左侧（268px）
- 主内容区自适应
- 最大内容宽度 960px
- 响应式断点：768px / 1024px

## 动画
- 状态变化使用 transition（0.2s ease-out）
- 页面切换使用 opacity + translateY
- 不使用 bounce/elastic 动画

## 禁止项
- 不用渐变文字
- 不用毛玻璃效果作为默认
- 不用侧边彩色边框
- 不用完全相同的卡片网格
